# likanu-xkb
xkb map for kokanu's likanu writing

# Installation
1. Download the file.
2. Place `kokanu` in `/usr/share/X11/xkb/symbols/` (on default arch linux, might be different on other distros)
3. You can now switch to it with `setxkbmap kokanu` (mak  sure you can switch back to you standard keyboard layout, otherwise you will have to reboot).

# Usage
- execute `setxkbmap us,kokanu -option grp:win_space_toggle` on startup
  - This will swap between us and kokanu layouts when Windows+Space is pressed.
